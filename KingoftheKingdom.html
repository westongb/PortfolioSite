King of the Kingdom was my first full stack web application. I was introduced to coding while working at Helio Training coding bootcamp where they taught the MERN stack. The instructors, mentors and Alumni there helped me to learn and build this first Web Application.

ReactJS

ReactJS is one of the most popular front end frameworks in development. So when it came time to build my first application Create React App was the best and easiest place to start.

Node, Express and MongoDB

I then started learning how to build a back end. I learned Express and Node to build my API end points to my MongoDB database with Mongoose middleware.

Stories and RoundTable

The site has two different applications. Each consists of a form that they fill out with different goals and objectives. These are stored in the database and then displayed for review as you work to acheive the goals you set.

User Login

Each user has to set up an account and the posts to their Story and Roundtable are stored under their user name. Passwords are stored in the database and are encrypted with Bcrypt and JSON Web Tokens.

Heroku SSR

The whole app is then compited for production and run through Server Side Rendering and hosted on Heroku.
